Lego Explorer

Welcome to Lego Explorer - your ultimate hub for exploring, tracking, and discovering Lego sets! This website serves as a SaaS platform for Lego enthusiasts of all ages. Whether you're a casual builder, serious collector, or anyone in between, we have something for everyone!

About the Project

Lego Explorer is part of a web development assignment designed to provide hands-on experience with both front-end and back-end development, RESTful principles, and cloud deployment on platforms like AWS, Azure, or GCP.

API Documentation

Our API provides access to Lego set data, supporting operations like retrieving, searching, and managing records in user collections. The API adheres to REST principles and is fully documented in Postman. You can access the documentation through the link provided in our README file.
